# essentials

## Project setup

```
# yarn
yarn
```

### Compiles and hot-reloads for development

```
# yarn
`yarn dev`

This concurrently runs the front and backend.

To independently run:
### Frontend
`yarn dev:frontend`

### Backend
`yarn dev:backend`

```

### Compiles and minifies for production

```
# yarn
yarn build
```

### Lints and fixes files

```
# yarn
yarn lint
```
